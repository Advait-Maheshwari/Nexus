# Nexus Design System

## Visual Language

Nexus should feel like premium mission-control software, not a marketing page. Use deep space surfaces, glass panels, restrained neon, precise typography, and cinematic motion.

## Palette

- Void: `#02040a`
- Deep Navy: `#06111f`
- Panel: `rgba(8, 17, 32, 0.68)`
- Cyan: `#48e5ff`
- Blue: `#4f8cff`
- Violet: `#8d67ff`
- Solar Gold: `#f5c451`
- Success Green: `#4ade80`
- Risk Red: `#fb7185`

## Interaction

- 3D is for orientation, exploration, and emotion.
- 2D glass UI is for editing and dense decisions.
- No landing-page detours inside the app.
- Motion should guide attention and preserve 60 FPS.

## Layout

- Mission Control is the default entry screen.
- Sidebar navigation remains predictable.
- Primary 3D scene is full-bleed behind work panels.
- Cards are reserved for repeated entities, metrics, and modals.
- Editing surfaces should be quiet, dense, and fast.
